## 📖 About the Website

**Gayathri Jewellers** is a trusted jewellery store based in **Chhatrapati Sambhaji Nagar, Maharashtra**, serving customers since **1995**. This website is a modern, elegant digital presence for the brand, built to showcase their exquisite jewellery collections online.

### 🌟 Key Highlights

| Feature | Details |
|---|---|
| 🏛️ Established | Since 1995 (28+ Years of Legacy) |
| 👥 Happy Clients | 50,000+ |
| 📍 Location | Chhatrapati Sambhaji Nagar, Maharashtra |
| 💰 Gold Standard | Certified 22K & 24K Gold |

## 💍 Website Sections

- **Home** — Elegant hero banner with *"Timeless Elegance For Your Only Moment"*
- **About** — Story of Gayathri Jewellers and their 28+ year heritage
- **Collections** — Curated jewellery collections including the Heritage Bridal Collection
- **Gallery** — Visual showcase of ornaments and designs
- **Contact** — Appointment booking and store information

- 🛡️ **Lifetime Guarantee** — Comprehensive insurance and lifetime maintenance for your precious ornaments

## 🚀 Run Locally

**Prerequisites:** Node.js

1. Clone the repository:
```bash
   git clone https://github.com/Parassanghavi1008/Gayatri_Jwellers_Website.git
   cd Gayatri_Jwellers_Website
```

2. Install dependencies:
```bash
   npm install
```

3. Run the development server:
```bash
   npm run dev
```
